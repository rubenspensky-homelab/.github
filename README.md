# Rubenspensky Homelab GitHub Metadata

This repository contains GitHub organization metadata for `rubenspensky-homelab`.

## Profile README

The organization profile content is maintained in [`profile/README.md`](./profile/README.md). GitHub renders that file on the organization overview page.

## Source Repositories

- [`homelab-infra`](https://github.com/rubenspensky-homelab/homelab-infra) defines platform infrastructure and cluster services.
- [`homelab-apps`](https://github.com/rubenspensky-homelab/homelab-apps) defines application Kubernetes manifests.
